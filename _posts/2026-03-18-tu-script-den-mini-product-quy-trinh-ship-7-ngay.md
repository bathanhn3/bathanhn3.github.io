## Ngay 1: Chot bai toan

Xac dinh 1 bai toan cu the va lap lai nhieu lan. Neu bai toan mo ho, 7 ngay se tro thanh 7 ngay loanh quanh.

## Ngay 2: Viet script core

Lam ra ban chay duoc cho 1 case don gian nhat.

## Ngay 3: Them input/output ro rang

- Input: file CSV, JSON, hoac argument CLI.
- Output: report CSV, markdown, hoac JSON.

## Ngay 4: Them xu ly loi

Script phai fail co kiem soat: timeout, input sai dinh dang, API loi.

## Ngay 5: Dong goi va viet README

README tra loi 3 cau hoi:

1. Tool giai quyet van de gi?
2. Chay nhu the nao?
3. Dau ra la gi?

## Ngay 6: Test voi du lieu that

Dung du lieu ngoai doi de lo edge case. Day la buoc quyet dinh.

## Ngay 7: Publish va xin feedback

Publish len GitHub, viet bai chia se ngan va moi 3-5 nguoi dung thu.

## Mau checklist ship nhanh

- [ ] Tool chay duoc tren may moi.
- [ ] Co sample input.
- [ ] Co sample output.
- [ ] Co changelog v0.1.
- [ ] Co issue list cho v0.2.
