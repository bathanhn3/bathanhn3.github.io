## Thang 1-2: Nen tang

Muc tieu khong phai hoc het, ma la hieu du de build.

- Bien, ham, dieu kien, loop.
- Cau truc du lieu co ban.
- Git can ban.
- HTTP va JSON.

Output can dat: 3 script nho giai quyet viec that.

## Thang 3-4: Lam quen voi du lieu va API

- Doc/ghi file CSV, JSON.
- Goi API bang Python.
- Xu ly loi (`try/except`, logging co ban).
- Viet script co tham so dau vao.

Output can dat: 1 workflow automation nho (VD: tu keyword list tao title + meta draft).

## Thang 5: Dong goi thanh tool

- Chia code thanh module.
- Tao CLI bang `argparse`.
- Them README, huong dan setup.

Output can dat: 1 mini tool co the giao cho nguoi khac dung.

## Thang 6: Build in public

- Viet 3 bai chia se: van de, cach giai, bai hoc.
- Refactor tool theo feedback.
- Theo doi metric su dung.

Output can dat: 1 mini-product duoc publish.

## Cach hoc de di duoc duong dai

### Timebox thay vi hoc hung

Moi ngay 60-90 phut, deu hon la hoc 1 lan 8 tieng roi nghi 1 tuan.

### Du an nho > tutorial dai

Tutorial giup bat dau nhanh, nhung project nho moi tao tri nho lau.

### Viet lai dieu ban vua hieu

Neu khong giai thich duoc bang loi cua minh, nghia la ban chua hieu du sau.
