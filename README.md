# bathanhn3.github.io

This site now uses Jekyll with clean SEO-friendly URLs.

## URL structure

- Homepage: `/`
- Blog listing: `/blog/`
- Blog post: `/blog/<slug>/`
- About: `/about/`
- Contact: `/contact/`

## Content structure

- Posts: `_posts/YYYY-MM-DD-slug.md`
- Layouts: `_layouts/`
- Includes: `_includes/`
- Styles: `css/`

## Local run

1. Install Ruby + Bundler.
2. Install dependencies:
   `bundle install`
3. Run local server:
   `bundle exec jekyll serve`
